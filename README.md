# xDownloader Bot (No Limit Version)

Bot Telegram untuk mengunduh video dari Twitter/X dan TikTok dengan kualitas terbaik *tanpa limit resolusi*.

## Fitur
- Unduh kualitas terbaik otomatis
- TikTok & Twitter/X
- Tanpa limit resolusi
- Otomatis merge audio/video
- Siap deploy ke Railway

## Deploy ke Railway
1. Upload file ke GitHub
2. Hubungkan repo ke Railway
3. Tambah variable:
   - `BOT_TOKEN`
4. Deploy
